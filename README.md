# Gravatá Suplementos - Sistema de Pedidos

Sistema simples de catálogo e pedidos pelo WhatsApp para a loja Gravatá Suplementos.

## Funcionalidades

- Catálogo de produtos por categoria
- Busca por produto
- Carrinho com quantidade e total automático
- Carrinho salvo no navegador com localStorage
- Formulário com nome, retirada/entrega, cidade e observação
- Envio do pedido organizado para WhatsApp
- Layout mobile-first com cores preto e laranja

## WhatsApp configurado

Número usado no projeto: `5581989282659`

## Como rodar

```bash
npm install
npm run dev
```

## Onde alterar produtos

Abra o arquivo:

```bash
src/main.js
```

Edite o array `productsData`.
